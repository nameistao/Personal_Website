## Project Name & Pitch

Personal Website

My personal website to show employment history, skills, and projects.

## Project Context

This project was for converting my old personal website, which was made using bootstrap, to this one, which uses react and next. I wanted to tools
used to build my latest personal website to reflect the technologies that I am currently using.

## Reflection

#### Unexpected Challenges:

- The main challenge with this project was design, which includes layout and color schemes. I wanted to create a design that allowed the user to easily find the information they were looking for.

#### What I Learned/Improved Upon:

- React
- Next
- Deploying the webapp
